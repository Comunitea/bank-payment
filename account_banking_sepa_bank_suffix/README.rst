Account Banking Sepa - Sepa identifier suffixes per payment mode
================================================================

* This module allows to change sepa identifiers suffixes from payment modes.

Installation
============

This module depends on : - account_banking_pain_base

This module is part of the OCA/bank-payment suite.

Configuration
=============

No configuration required.

Usage
=====

* Creates a suffix field on payment modes, if this field is set, it replace the suffix configured in sepa identifiers of company.
